fixedMate

is a sass based code, spiced with javascript. It was made for people who (like I before) have problems with fixed slider divs. They are everywhere: social plugins, email subscriptions or ads. My idea was to make it as simple as it can be, so all a user has to do is add a class to his div.

What class should I add?

use:

class="fixed-mate-left"
to create a left floated fixedMate and use:

class="fixed-mate-right"
if you want to create a right floated fixedMate

that is all... if you want a 300 x 300 px fixedMate, if not fear not. There's an easy way to change it.

I want to make it bigger and wider

You don't need to know sass to change width and height. Go to style/main.scss and change:

$mate-height: 300px; // your fixedMate height $mate-width: 300px; // your fixedMate width

Now give them a try.
